### ADOPTION PROCESS (per Lane, 28 Mar 2026)
> The Southern Nevada Building Officials (SNBO) adopt versions of the codes together, then an ordinance is passed by each political subdivision of the state to tweak the codes for local needs.
> In Utah, the state adopts the codes (state-level, not local).
> Local development codes also exist in both southern and northern Nevada — these layer ON TOP of the base building code.
>
> **Data source for effective dates:** Query SNBO directly, or pull from Clark County Development Services official adoption records. Each jurisdiction's ordinance may have a slightly different effective date even though SNBO coordinates the adoption.
