Forensic Noir × Dannaway Craft · v1.0

A forensic instrument, not a SaaS dashboard.

BlackBar is the UI layer over a four-agent pipeline that takes files, notes, and photos and returns a finished SCG Initial Report. Every surface is agent-honest: you see what the agent sees, what it found, and what it can't. No decoration, no black boxes, no pretending a mocked state is a real one.

01 · Palette

Color, chosen on purpose.

No pure black, no pure white, no gradient. Amber is used like a scalpel — on the primary CTA, the live state, the focus ring, and one finding-highlight per surface. That's it.

02 · Typography

Fraunces + Inter Tight.

Fraunces (optical-sized serif) carries display authority — case titles, section headers, scorecard numerals. Inter Tight handles UI and body. JetBrains Mono for timestamps, token counts, and JSON. No default Inter anywhere.

10 · Inline Flags

Visual language for truth.

The system prompt emits flags; the UI renders them. Every flag tells the reader something the agent couldn't hide.

agent blind
Surface exists, agent doesn't reason over the content yet. Non-negotiable for stubbed features.
compliance review
Output touches regulated or precedent-cited language. Mariz eyes before send.
privileged
Touches Cases/ or benchmarks/. Never commit without explicit request.
live
Agent currently streaming via SSE. Dot pulses.
complete
Stage or check passed. Green by design.
failed
Blocking error. Halt condition.
